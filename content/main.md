## About This Project

This is a demonstration of API-based content protection using GitHub OAuth and the GitHub Contents API.

## How It Works

1. You authenticate with GitHub
2. Your OAuth token is used to fetch this content from the private repository
3. GitHub validates your permissions server-side
4. Only users with repository access can view this content

## Your Access

Your GitHub token provides access to this private repository content. The token is scoped to your GitHub permissions and expires according to GitHub's OAuth policies.
