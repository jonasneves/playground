# Prototype Gallery Framework

Generic React framework for building prototype galleries with GitHub integration.

## Available Files

- **framework.es.js** - ES Module build (10 KB gzipped)
- **framework.umd.js** - UMD build (9 KB gzipped)
- **styles/global.css** - Default styles

## Components

- OAuth - GitHub authentication
- AppShell - Main gallery interface
- AppCard - App display cards
- UserMenu - User dropdown menu

## Utilities

- createAuthStore - Authentication state management
- createCacheStore - API caching
- createAnalyticsStore - Event tracking
- createGitHubAPIHook - GitHub API integration

## Security

This framework contains only generic building blocks.
No sensitive data or private implementation logic is included.
